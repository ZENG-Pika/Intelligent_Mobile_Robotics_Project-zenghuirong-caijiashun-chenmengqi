# Intelligent_Mobile_Robotics_Project-zenghuirong-caijiashun-chenmengqi
## 📖 项目简介 (Introduction)

本项目是浙江大学《智能移动机器人》课程的大作业实验，小组成员： **曾辉荣， **蔡嘉顺， **陈梦奇

项目旨在解决复杂三维环境下的移动机器人自主导航问题。系统采用了分层运动规划框架：
1.  **前端路径规划 (Path Planning)**：利用 **A* (A-Star)** 算法在三维栅格地图中搜索出无碰撞的全局最优几何路径。
2.  **后端轨迹生成 (Trajectory Generation)**：利用 **三次样条插值 (Cubic Spline Interpolation)** 对离散路径进行平滑处理，生成满足 $C^2$ 连续性（位置、速度、加速度连续）的时间参数化轨迹。

## ✨ 主要功能 (Features)

* **三维环境仿真**：能够生成指定数量的随机圆柱体障碍物，模拟非结构化复杂环境。
* **A* 全局搜索**：基于 26-邻域扩展和欧几里得启发函数的 A* 算法，能够高效找到避障路径。
* **轨迹平滑**：基于弦长参数化的三次样条插值，消除路径折点，生成平滑曲线。
* **鲁棒性处理**：包含对路径重叠点的预处理机制，防止插值数值错误。
* **可视化展示**：
    * 3D 视图：显示障碍物、起点、终点及生成的路径。
    * 时间响应视图：展示 $x(t), y(t), z(t)$ 随时间变化的平滑曲线及原始路径点对比。
